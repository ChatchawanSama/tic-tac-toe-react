#Check node and npm version 

1. Check node is already have?
```sh
node -v
```
2. Check npm is already have?
```sh
npm -v
```
3. Install yarn
```sh
npm install -g yarn    
```
4. Check yarn is already have?
```sh
yarn -v 
```
5. Install package.json
```sh
yarn install
```
6. Run to Start Server 
```sh
yarn start
```
